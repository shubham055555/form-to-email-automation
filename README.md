# 📩 Form to Email Automation

This project is an automation workflow that sends a confirmation email when a user submits a form.

## 🚀 Features
- Google Form integration
- Data stored in Google Sheets
- Automatic email sent to user
- Dynamic fields (Name, Service, Time)

## ⚙️ Tech Stack
- n8n / Make / Zapier
- Gmail API
- Google Sheets

## 📂 Workflow File
- `workflow.json`

## 🧠 How it Works
1. User fills form
2. Data stored in Google Sheets
3. Automation triggers
4. Email sent to user
